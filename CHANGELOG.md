# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-10

### Added
- Initial release of Secret PDF SDK
- `SecretPDFClient` class for easy API interaction
- Full TypeScript support with type definitions
- Support for all Secret PDF API endpoints:
  - Document generation from HTML templates
  - Template management
  - API key management
  - Storage configuration
  - Webhook management
  - User authentication and billing
  - Usage statistics and analytics
- Comprehensive documentation and examples
- ESM module support
- Built-in error handling

### Features
- Generate PDFs from HTML with template variables
- Store PDFs in cloud storage (S3, etc.)
- Create and manage reusable templates
- OAuth2 and API key authentication
- Full access to billing and usage data
- Webhook support for event notifications
